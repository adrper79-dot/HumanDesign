## 🚨 Alert Triggers

**CRITICAL (Immediate Action):**
- API error rate exceeds 5% for >5 minutes
- Chart generation fails >3 times in a row for same user
- CSS file 404 (CDN delivery failed)
- All users see same error on profile page

**HIGH (Investigate within 1 hour):**
- Chart generation time spikes >5 second average
- Assessment completion rate drops >20% vs baseline
- Error rate >2% for >30 minutes
- Onboarding abandonment >60% (vs target 40%)

**MEDIUM (Log and review next day):**
- Tooltip hover rate <10% (users not discovering new feature)
- Savannah Arc feature not mentioned in support tickets
- Copy clarity metrics (scroll depth) below expectations
