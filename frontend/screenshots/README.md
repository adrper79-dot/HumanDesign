1. **Home/Dashboard** - User profile, tier badge, navigation
2. **Bodygraph Chart** - Full HD chart with all centers visible

### Design Requirements
- Use actual app interface (capture from running app)
- Show populated data (not empty states)
- Use premium tier features for best impression
- High quality PNG format
- Clean, professional appearance

## Capture Instructions

### For Mobile Screenshots (540x720)
1. Open Prime Self in Chrome DevTools
2. Set device emulation to 540x720 resolution
3. Navigate to desired screen
4. Take screenshot with DevTools screenshot tool
5. Save as `home.png` in this directory

### For Desktop Screenshots (1280x720)
1. Open Prime Self in browser
2. Resize window to 1280x720
3. Navigate to chart view
4. Use browser screenshot extension or DevTools
5. Save as `chart.png` in this directory
